# Data Format & Column Schema

## Input File
Expected format: Excel (`.xlsx`) or CSV (`.csv`)

---

## Required Columns

| Column | Type | Unit | Source | Description |
|--------|------|------|--------|-------------|
| `STATE_KEY` | str | — | Agmarknet | State name identifier (e.g., `"Tamil Nadu"`) |
| `DATE_STD` | date | DD/MM/YYYY | Agmarknet | Market observation date |
| `Modal Price` | float | ₹/quintal | Agmarknet | Modal (most frequent) price |
| `Min Price` | float | ₹/quintal | Agmarknet | Minimum daily price |
| `Max Price` | float | ₹/quintal | Agmarknet | Maximum daily price |

---

## Climate Columns (NASA POWER API)

| Column | Unit | Description |
|--------|------|-------------|
| `T2M_MAX` | °C | Maximum air temperature at 2 metres |
| `T2M_MIN` | °C | Minimum air temperature at 2 metres |
| `T2M_RANGE` | °C | Diurnal temperature range (T2M_MAX − T2M_MIN) |
| `PRECTOTCORR` | mm/day | Bias-corrected total precipitation |
| `RH2M` | % | Relative humidity at 2 metres |
| `WS2M` | m/s | Wind speed at 2 metres |
| `ALLSKY_SFC_SW_DWN` | MJ/m²/day | All-sky surface shortwave downward irradiance |
| `PSC` | % | Percentage of sunshine |

---

## Agricultural Columns (Ministry of Agriculture, India)

| Column | Unit | Description |
|--------|------|-------------|
| `PRODUCTION_TONNES` | tonnes | Annual state-level rice production |
| `AREA_HA` | hectares | Net sown area under rice |
| `YIELD_TON_PER_HA` | t/ha | Yield per unit area |

---

## Engineered Features (auto-generated by Step 3)

These are created automatically by `engineer_features()` and do **not** need to be present in the raw input:

| Feature | Description |
|---------|-------------|
| `price_lag_1/3/7/14/30` | Lagged modal price (t−k days) |
| `price_ma_7/14/30` | Rolling mean price |
| `price_std_7/14/30` | Rolling std of price |
| `month`, `quarter`, `weekofyear`, `dayofyear` | Calendar features |
| `month_sin`, `month_cos` | Cyclical month encoding |
| `GDD` | Growing Degree Days = max(Tmean − 10°C, 0) |
| `T2M_MAX_lag_7/14` | Lagged maximum temperature |
| `T2M_MIN_lag_7/14` | Lagged minimum temperature |
| `PRECTOTCORR_lag_7/14` | Lagged precipitation |
| `Price_Change_Percent` | % change from previous observation |
| `y` | Log-transformed target: log1p(Modal Price) |

## Sample Data

`sample_data.csv` contains 500 rows with all required raw columns populated with  
anonymised/synthetic values that preserve the statistical properties of the original dataset.  
It is suitable for smoke-testing the pipeline and verifying installation.

- **Agricultural statistics**: [MOSPI / Directorate of Economics & Statistics](https://desagri.gov.in/)
